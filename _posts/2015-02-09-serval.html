Serval is an Open Source australian project that provides a phone-to-phone mesh network.

In a mesh network, devices to communicate with each other in a distributed fashion, bypassing a central authority.

Serval Chat is a user-friendly app version of Serval Mesh. Since the more nodes the better the connection, reducing friction is crucial.

Goals

  • Consistent experience across phone/tablet/desktop
  • Conversation methods are text, file, image, voice message (30" limit), video (15" limit) with possible plugins.
  • Easy conversation retrieval.

Contacts Panel

Contacts panel cycles Friends (contacts both agreed to talk), Groups (groups user belongs to) and Requests (permissions asked and requested)
